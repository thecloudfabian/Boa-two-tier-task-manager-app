# BOA Two-Tier AWS Task Manager App

## Overview
A scalable two-tier web application deployed on AWS using EC2, RDS, and an Application Load Balancer.

## Architecture
- EC2 Instances (Node.js app layer)
- Amazon RDS MySQL (Database Layer)
- Application Load Balancer (Traffic Distribution)
- VPC with Public/Private Subnets

## Features
- Add tasks to database
- Persistent storage using MySQL (RDS)
- Load-balanced across multiple EC2 instances
- High availability across AZs

## Services Used
- Node.js
- Express.js
- MySQL (Amazon RDS)
- AWS EC2
- AWS ALB
- AWS VPC
